# Experimentum
An interactive educational game for performing virtual science experiments.

## Overview
This application allows users to perform virtual science experiments in a safe and interactive environment. Each experiment provides step-by-step guidance and explains the science behind it.

## Features
- 12 interactive science experiments
- Step-by-step task system
- Real-time feedback for correct and incorrect actions
- Progress saving and experiment unlocking

## How to Play
1. Launch the application  
2. (Optional) Register or log in to save progress  
3. Select an experiment  
4. Follow the on-screen to-do list  
5. Complete tasks in the correct order  

## Controls
- Mouse: Interact and drag objects  
- ESC: Pause the game  

## Game Flow
- Complete all tasks before time runs out  
- Incorrect actions result in time penalties  
- Successfully finishing an experiment plays a reaction video and unlocks the next one  

## Notes
- Progress is saved automatically after completing experiments  
- Restart the experiment if unexpected behavior occurs
